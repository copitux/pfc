model_base.class.php - Modelo base

Modelo del que heredan todos los modelos.  Clase abstracta donde se implemente el patron 'active record' y se realiza la conexion a la base de datos
Cada modelo tendra los datos de la tabla correspondiente en su base de datos. En esta clase se definen los  metodos necesarios para el dinamismo modelo-DB.  A su vez se instancia el objeto PDO, encargado de la conexion a la DB.
